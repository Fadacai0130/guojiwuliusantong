📋 隐私政策

国际物流查询 Chrome 扩展 · 最后更新：2026年2月

本扩展（"国际物流查询 - 三通国际物流"）尊重并保护用户隐私。本政策说明我们如何处理您的数据。

数据收集

本扩展不收集、不传输任何个人可识别信息（PII）。我们不设置追踪 Cookie，不向任何第三方分析服务发送数据。

本地存储

扩展使用 chrome.storage.sync 在您的 Chrome 账户中同步保存以下配置：

① 您填写的 API Endpoint（可选）；② API Key（可选）；③ 最近查询的运单号（最多 5 条，仅存于 chrome.storage.local）。

以上数据仅用于实现查询功能，不会被开发者读取或上传。

第三方 API

若您在设置中填写了自定义查询 API，扩展将向该地址发送运单号和承运商信息以获取物流状态。该第三方服务的数据处理由其自身隐私政策约束，与本扩展无关。

网络权限

扩展仅在您主动点击"查询"时，向您配置的 API 地址发起网络请求。未配置 API 时，不发起任何网络请求。

安全建议

请勿在公共或共享设备上保存敏感 API Key。如需撤销，可在扩展设置中点击"清除"。
